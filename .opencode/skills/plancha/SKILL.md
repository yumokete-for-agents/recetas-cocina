# Cocción a la plancha — Sartén plana, mínima grasa

## Principios fundamentales

- **Plancha bien caliente antes de añadir nada**: si no chisporrotea al poner el ingrediente, no está lista.
- **Aceite mínimo**: pincelar, no bañar. La plancha antiadherente necesita muy poca grasa.
- **No mover demasiado pronto**: los ingredientes se sueltan solos cuando están dorados. Si se pegan, no están listos.
- **Temperatura por ingrediente**: no todo se cocina al máximo. Carnes y pescados finos necesitan fuego alto; verduras y huevos requieren control.
- **No sobrecargar**: la plancha se enfría si metes demasiado de golpe. Cocinar en tandas.
- **Secar siempre la superficie**: la humedad impide el dorado. Secar con papel de cocina antes de poner en la plancha.

## Carnes

### Filetes finos (entrecot, chuleta, lomo — 1-2 cm)

| Variable | Valor |
|---|---|
| Temp plancha | Alta (250-300 °C) |
| Aceite | Pincelar la carne, no la plancha |
| Tiempo | 2-3 min x lado (poco), 3-4 min x lado (punto) |
| Señal | Dorado formado, soltarse solo. Dar la vuelta 1 vez |
| Reposo | 3-5 min fuera, tapado ligeramente |

### Filetes gruesos (steak 3-5 cm)

| Variable | Valor |
|---|---|
| Temp plancha | Muy alta al inicio, bajar a media |
| Aceite | 1 cdta aceite en la plancha + pincelar carne |
| Tiempo | 4-6 min x lado (punto) |
| Técnica | Sellar 2 min x lado a fuego máximo, luego bajar y cocer 2-3 min más x lado |
| Reposo | 5-8 min tapado |

### Tiras / dados de pollo, cerdo, ternera (brochetas, salteados)

| Variable | Valor |
|---|---|
| Temp plancha | Alta (250 °C) |
| Aceite | 1 cdta en la plancha |
| Tiempo | 3-4 min total (mover constante) |
| Señal | Dorado exterior, cocido interior |
| Corte | Tiras finas (~1 cm), dados uniformes (~2 cm) |

### Hamburguesas (100-150 g)

| Variable | Valor |
|---|---|
| Temp plancha | Alta (280 °C) |
| Aceite | Pincelar la hamburguesa |
| Tiempo | 3-4 min x lado (punto) |
| Señal | Costra formada, soltarse sola |
| Truco | No aplastar con la espátula — se exprimen los jugos |

### Hígado / vísceras (higado, mollejas, riñones)

| Variable | Valor |
|---|---|
| Temp plancha | Alta (250 °C) |
| Aceite | 1 cdta aceite |
| Tiempo | 2-3 min x lado |
| Señal | Dorado fuera, rosado dentro (higado no debe quedar gris) |

## Pescados

### Lomos con piel (salmón, lubina, dorada, merluza)

| Variable | Valor |
|---|---|
| Temp plancha | Alta (250 °C) |
| Aceite | Pincelar el lomo, especialmente la piel |
| Tiempo | 70% tiempo lado piel: 4-5 min. Girar y 1-2 min lado carne |
| Punto | Piel crujiente, centro traslúcido/opaco según gusto |
| Señal | La piel se dora y cruje. La carne sube de color desde abajo |
| Truco | Poner el lomo piel abajo y presionar 10 seg con espátula para que la piel selle plana |

### Lomos sin piel / medallones (rape, bacalao, merluza)

| Variable | Valor |
|---|---|
| Temp plancha | Media-alta (220 °C) |
| Aceite | 1 cdta aceite en la plancha |
| Tiempo | 3-4 min x lado según grosor |
| Punto | Opaco, se desmenuza ligeramente |

### Pescados pequeños enteros (sardinas, boquerones, salmonetes)

| Variable | Valor |
|---|---|
| Temp plancha | Alta (280 °C) |
| Aceite | Pincelar cada pieza |
| Tiempo | 2-3 min x lado |
| Señal | Piel crujiente, carne blanca/opaca |
| Truco | Hacer cortes transversales en el lomo para que se cocinen parejo |

### Atún / pez espada (medallones 2-3 cm)

| Variable | Valor |
|---|---|
| Temp plancha | Muy alta (280 °C) |
| Aceite | Pincelar el medallón |
| Tiempo | 2-3 min x lado (poco, centro rojo) |
| Señal | Dorado exterior, centro rojo jugoso |

## Mariscos

### Gambas / langostinos (con cáscara o pelados)

| Variable | Valor |
|---|---|
| Temp plancha | Muy alta (300 °C) |
| Aceite | 1 cdta aceite |
| Tiempo | 2-3 min total (con cáscara 1.5-2 min x lado) |
| Señal | Cáscara dorada/rosada, carne opaca |
| Truco | Con cáscara: más sabor, más jugosas. Peladas: cuidado de no pasarse |

### Calamares / sepia (anillas o cuerpos)

| Variable | Valor |
|---|---|
| Temp plancha | Muy alta (300 °C) |
| Aceite | 1 cdta aceite |
| Tiempo | 2-3 min total |
| Señal | Dorado, tierno. Si pasa de punto se pone gomoso |
| Truco | Cocer muy rápido a fuego máximo. O cocción muy lenta (30 min) — el punto medio es el error |

| Aceite | Pincelar la concha |
| Tiempo | 1-2 min x lado |
| Señal | Doradas, centro traslúcido |

### Pulpo (previamente cocido)

|---|---|

| Variable | Valor |

## Huevos

### Huevo plano (frito / a la plancha)

| Variable | Valor |
|---|---|
| Temp plancha | Media (180 °C) |
| Aceite | ½ cdta aceite o mantequilla |
| Tiempo | 3-4 min (clara cuajada, yema líquida) |
| Señal | Clara blanca firme, bordes crujientes |
| Truco | Tapar 30 seg al final para que cuaje la clara sin quemar la yema |

### Huevo revuelto

| Variable | Valor |
|---|---|
| Temp plancha | Media-baja (160 °C) |
| Aceite | ½ cdta mantequilla |
| Tiempo | 3-5 min, mover constante |
| Técnica | Mover con espátula cada 15-20 seg, retirar cuando aún cremoso |

### Tortilla de patatas (cuajada en plancha)

| Variable | Valor |
|---|---|
| Temp plancha | Media (180 °C) |
| Aceite | 2 cdas aceite de oliva |
| Tiempo | 12-15 min una cara, 6-8 min la otra (dar la vuelta con plato) |

## Avisos de seguridad

| Ingrediente | Advertencia |
|---|---|
| **Pollo / pavo** | Cocer hasta **74 °C** en el centro. No servir rosado ni poco hecho. La plancha dora rápido por fuera pero el centro puede quedar crudo — reducir grosor o aplastar ligeramente. |
| **Carne picada** (hamburguesas) | Cocer hasta **71 °C** en el centro. No servir poco hecho. |
| **Huevos** (yema cruda) | Advertir a embarazadas, niños, ancianos e inmunodeprimidos si se sirve yema líquida. Usar huevos pasteurizados si hay duda. |
| **Contaminación cruzada** | Limpiar la plancha entre tandas de carne cruda y otros ingredientes. Usar espátulas diferentes. |

## Micro-ajustes

| Ajuste | Cómo se hace |
|---|---|
| **Más dorado / costra** | Plancha más caliente, secar bien superficie, no mover |
| **Más jugoso** | No sobrecocer, reposar, fuego alto para sellar rápido |
| **Menos aceite** | Usar plancha antiadherente de calidad. Pincelar ingrediente, no la plancha |
| **Más crujiente (piel pescado)** | Salar la piel y dejar 10 min abierta. Cocer 80% tiempo lado piel. Presionar al inicio |
| **Sabor extra** | Marinar 15-30 min. Añadir especias al aceite de pincelar. Toque de salsa al servir |
| **Color perfecto** | No sobrecargar la plancha. Temperatura constante entre tandas. Esperar a que recupere calor |

## Errores comunes

- **Plancha no suficientemente caliente**: los ingredientes se pegan y cuecen en su propio jugo en lugar de dorarse.
- **Mover la comida constantemente**: impide que se forme la costra dorada. Dejar quieto hasta que suelte solo.
- **Poner ingredientes húmedos**: la humedad impide el dorado. Secar siempre con papel de cocina.
- **Aceite en exceso**: la comida fríe en lugar de cocerse a la plancha. El aceite debe ser mínimo.
- **Aceite en la plancha y no en el ingrediente**: el aceite en la plancha se quema y humea. Mejor pincelar el ingrediente.
- **Sobrecargar la plancha**: la temperatura baja y todo se cuece al vapor. Cocinar en tandas.
- **Usar utensilios metálicos en antiadherente**: rayan la superficie. Usar madera o silicona.
- **Dar la vuelta al pescado demasiado pronto**: se rompe. Esperar a que la piel esté crujiente y suelte naturalmente.
